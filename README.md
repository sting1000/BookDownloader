# 📚 电子书下载器 (BookDownloader)

一个简单的 macOS 应用，用于搜索和下载 EPUB 电子书。

## 功能

- 🔍 通过书名搜索 GitHub 上的 EPUB 电子书
- 📥 一键下载到本地
- 📖 下载完成后可直接打开阅读
- 🎨 原生 macOS 对话框界面

## 安装

### 方法一：直接使用
1. 下载本项目
2. 双击 `BookDownloader.app` 运行
3. 首次运行需要右键 → 打开 → 确认打开（未签名应用）

### 方法二：命令行
```bash
python3 book_downloader.py
```

## 使用方法

1. 打开应用
2. 在弹出的对话框中输入书名
3. 从搜索结果中选择想要的书籍
4. 选择保存位置
5. 等待下载完成
6. 选择是否立即打开阅读

## 系统要求

- macOS 10.13 或更高版本
- Python 3.6+（系统自带）

## 注意事项

- 本工具仅搜索 GitHub 上公开的电子书资源
- 请尊重版权，仅下载合法分享的书籍
- 如果喜欢某本书，请支持正版
